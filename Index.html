WORKLEDGER
GITHUB DEPLOYMENT GUIDE
This guide deploys your WorkLedger app to the internet using GitHub Pages — completely free.
Your app will live at a permanent URL and your data will save in the browser automatically. No servers, no subscriptions.
```
1
Download the project ZIP
Download the workledger.zip file (provided alongside this guide) and unzip it on your computer. You'll see a folder called workledger with these files inside:
workledger/ ``` ├── index.html ├── package.json ├── vite.config.js ├── src/ │ ├── App.jsx │ └── main.jsx └── .github/ └── workflows/ └── deploy.yml
```
2
Create a free GitHub account
  1. Go to github.com
  2. Click Sign up and create your account
  3. Verify your email address
Already have a GitHub account? Skip to Step 3.
3
Create a new repository
  1. Click the + icon (top-right corner) → New repository
  2. Set the repository name to exactly: workledger
  3. Make sure it is set to Public
  4. Leave everything else as default
  5. Click Create repository
4
Upload all project files
  1. In your new empty repository, click "uploading an existing file"
  2. Open the unzipped workledger folder on your computer
  3. Select all files and folders inside it and drag them into the GitHub upload area
  4. Wait for all files to appear in the list
  5. Scroll down and click Commit changes
⚠️ Make sure to upload the .github folder too — it's hidden on Mac/Windows but required for auto-deployment. On Mac, press Cmd+Shift+. to show hidden files before dragging.
5
Enable GitHub Pages
  1. Go to your repository → click Settings (top tab)
  2. In the left sidebar, click Pages
  3. Under Source, select GitHub Actions
  4. Click Save
✅ That's it! GitHub will now automatically build and deploy your app every time you upload changes.
6
Wait ~2 minutes for first deploy
  1. Go to your repository → click the Actions tab
  2. You'll see a workflow running called "Deploy WorkLedger"
  3. Wait for the yellow circle to turn into a green checkmark ✅
First deployment takes 1–3 minutes. After that, updates deploy in under 30 seconds.
```

https://YOUR-USERNAME.github.io/workledger

Replace YOUR-USERNAME with your actual GitHub username
